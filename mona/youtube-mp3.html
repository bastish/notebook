How to Download and Convert YouTube Videos to MP3 on macOS

Part 1: Install yt-dlp

1. Download yt-dlp

  1. Open this link: Download yt-dlp_macos
  2. Save the file in your Downloads folder.

2. Move yt-dlp to Applications

  1. Open Finder and go to your Downloads folder.
  2. Drag the file named yt-dlp_macos into your Applications folder.

3. Allow macOS to Open yt-dlp

  1. Open Finder and go to your Applications folder.
  2. Locate yt-dlp_macos.
  3. Right-click (or Control-click) the file and select Open.
  4. A pop-up will appear saying: ‚Äúyt-dlp_macos can‚Äôt be opened because Apple cannot check it for malicious software.‚Äù Click Open Anyway.

Part 3: Download YouTube Videos as MP3

1. Move to the Downloads Directory in Terminal

Before running yt-dlp, move to the Downloads folder so your files are saved there:

cd ~/Downloads

2. Run yt-dlp

To download and convert a YouTube video to MP3, use this command:

/Applications/yt-dlp_macos --extract-audio --audio-format mp3 --ffmpeg-location /Applications/ffmpeg 'https://www.youtube.com/watch?v=VIDEO_ID'
  • Replace VIDEO_ID with the actual video link (e.g., https://www.youtube.com/watch?v=aNSH8OdHx2A).

Part 4: Adjusting MP3 File Size and Quality

üéµ How to Make the MP3 Smaller or Bigger

You can adjust the file size by modifying the bitrate during conversion:

  • Smaller File Size: Use a lower bitrate (e.g., 64kbps). Ideal for audiobooks or podcasts.
  • Higher Quality: Use a higher bitrate (e.g., 192kbps or 320kbps) for better sound but larger files.

üõ† Command Examples:

  • Low Quality (Small File, ~3MB): 
    /Applications/yt-dlp_macos --extract-audio --audio-format mp3 --audio-quality 64 --ffmpeg-location /Applications/ffmpeg 'https://www.youtube.com/watch?v=VIDEO_ID'
  • Standard Quality (Default, ~6MB): 
    /Applications/yt-dlp_macos --extract-audio --audio-format mp3 --audio-quality 128 --ffmpeg-location /Applications/ffmpeg 'https://www.youtube.com/watch?v=VIDEO_ID'
  • High Quality (Larger File, ~12MB): 
    /Applications/yt-dlp_macos --extract-audio --audio-format mp3 --audio-quality 320 --ffmpeg-location /Applications/ffmpeg 'https://www.youtube.com/watch?v=VIDEO_ID'

Part 6: Extra Features

Download MP4 Video Instead of MP3

To save the full video in MP4 format:

/Applications/yt-dlp_macos -f 'best[ext=mp4]' 'https://www.youtube.com/watch?v=VIDEO_ID'

Download Entire Playlists

To download all videos in a playlist:

/Applications/yt-dlp_macos 'https://www.youtube.com/playlist?list=PLAYLIST_ID'

Batch Download Using a List

Save URLs in a file (e.g., urls.txt) and use:

/Applications/yt-dlp_macos -a urls.txt
